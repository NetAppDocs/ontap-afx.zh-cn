= 创建 AFX 存储系统 SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以创建 SVM 来提供隔离并提高安全性。您可以针对组织内的不同团体或项目执行此操作。

.关于此任务
创建 SVM 时，您必须提供名称并配置至少一个用于客户端访问的协议。选择客户端协议后，系统还将提示您进行网络配置。创建 SVM 配置后，您可以根据需要更改它。

.开始之前
您需要以下物品：

* 至少四个 IP 地址
* IP空间名称


.步骤
. 在系统管理器中，选择*集群*，然后选择*存储虚拟机*。
. 选择image:icon_add_blue_bg.png["蓝色矩形框内有一个加号，后面跟着白色字母“add”"]。
. 为 SVM 提供一个名称。
. 选择客户端访问的协议并提供适当的配置详细信息。
. 为 SVM 添加网络接口，包括 IP 地址和子网掩码。
. 在*存储虚拟机管理*下，可选：
+
.. 启用最大容量并选择一个值
.. 管理 SVM 的管理员帐户


. 选择*保存*。
