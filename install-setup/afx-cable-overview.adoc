== 支持的 AFX 1K 布线配置

AFX 1K 存储系统的初始配置支持至少四个控制器节点通过双交换机连接到存储磁盘架。

附加控制器节点和磁盘架扩展了初始 AFX 1K 存储系统配置。扩展的 AFX 1K 配置遵循与下图所示的相同的基于交换机的布线方法。

image:../media/drw_afx_cable_overview_half_node_ieops-2358.svg["AFK 1K 存储系统示例布线"]
