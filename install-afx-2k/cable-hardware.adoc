= 用缆线连接AFX 2K存储系统的硬件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
为 AFX 2K 存储系统安装机架硬件后，为控制器安装网络电缆，并连接控制器与存储架之间的电缆。

.开始之前
有关将存储系统连接到网络交换机的信息，请联系您的网络管理员。

.关于此任务
* 这些程序显示了常见的配置。具体的布线取决于您为存储系统订购的组件。有关全面的配置详细信息和插槽优先级，请参阅link:https://hwu.netapp.com["NetApp Hardware Universe"^]。
* AFX 2K 控制器上的 I/O 插槽编号为 1 到 11。

* 布线图形显示箭头图标，指示将连接器插入端口时电缆连接器拉片的正确方向（向上或向下）。
+
插入连接器时，您应该感觉到它咔哒一声到位；如果没有感觉到咔哒一声，请将其取出，翻转并重试。

+
image:../media/drw_cable_pull_tab_direction_ieops-1699.svg["电缆拉片方向"]

+
[NOTE]
====
连接器组件很精密，安装到位时应小心。

====
* 当布线到光纤连接时，先将光纤收发器插入控制器端口，然后再布线到交换机端口。
* AFX 2K 存储系统使用 400GbE 电缆。400GbE 连接在控制器和交换机之间建立。存储架和交换机之间的连接使用 4x100GbE 分支电缆，100GbE 连接连接到驱动器架端口。
+
机架存储连接可以连接到交换机上的任何 100GbE 分支端口。HA/集群连接可以连接到交换机上的任何 400GbE 端口。所有"a"控制器端口都连接到交换机 A，所有"b"控制器端口都连接到交换机 B。





== 步骤 1：将控制器连接到管理网络

将每个控制器上的管理（扳手）端口连接到任一管理交换机，或将其直接连接到您的管理网络。

使用 1000BASE-T RJ-45 电缆将每个控制器上的管理（扳手）端口连接到管理网络交换机。

image::../media/oie_cable_rj45.png[RJ-45 电缆]

*1000BASE-T RJ-45 电缆*

image::../media/drw_afx_2k_management_connection_ieops-2863.svg[连接到您的管理网络]


IMPORTANT: 请勿插入电源线。

.步骤
. 连接到管理网络。

== 步骤 3：对集群和 HA 连接进行布线

使用 Cluster 和 HA 互连电缆将端口 e1a 和 e2a 连接到交换机 A，将端口 e1b 和 e2b 连接到交换机 B。e1a/e1b 端口用于 HA 连接，e2a/e2a 端口用于集群连接。

.步骤

*** e2a (集群)

*** e1b（HA）
*** e2b (Cluster)
+
*400GbE 电缆*
